# TaskTalk - しゃべるだけでやること整理

音声で話した内容をAIで自動解析してタスクリスト化する、シンプルでモダンなタスク管理アプリです。

話すだけでタスクリストを完全自動作成、あとはどんどんこなすだけ。もう無理！と思ったらまずTaskTalkに吐き出そう。

## 🚀 ユーザーへの提供方法

### **方法1: GitHub Pages でホスティング（推奨）**

**無料で誰でもアクセス可能なURLを提供できます：**

1. **GitHubリポジトリを作成**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/your-username/tasktalk.git
   git push -u origin main
   ```

2. **GitHub Pagesを有効化**
   - Settings → Pages
   - Source: `main` ブランチ選択
   - Save

3. **公開URL**
   - `https://your-username.github.io/tasktalk/`
   - このURLをユーザーに共有

**データの保存：**
- ✅ ローカルストレージに保存（ブラウザ内）
- ✅ ブラウザを閉じても消えない
- ✅ 同じブラウザなら何度でもアクセス可能
- ⚠️ ブラウザのデータ削除で消える
- ⚠️ 別のデバイス・ブラウザでは同期されない

### **方法2: PWA（Progressive Web App）としてインストール**

ユーザーがアプリとしてインストールできます：

#### 📱 **スマホ（iOS/Android）**
1. SafariまたはChromeでURLを開く
2. 「ホーム画面に追加」をタップ
3. ネイティブアプリのように使える

#### 💻 **PC（Chrome/Edge）**
1. URLを開く
2. アドレスバー右のインストールアイコン
3. デスクトップアプリとして起動

**PWA化に必要な追加ファイル（今後実装可能）：**
- `manifest.json` - アプリの設定
- `service-worker.js` - オフライン対応
- アイコン画像

### **方法3: Publishタブからデプロイ**

このエディタの「Publish」タブから：
1. ワンクリックでデプロイ
2. 自動でURLが発行される
3. すぐに共有可能

### 🤖 **ChatGPT APIによる高精度タスク抽出**
- Web Speech APIで日本語音声認識
- ChatGPT APIで自然な会話からタスクを正確に抽出
- 録音停止と同時に即座にタスクリスト化
- APIキーがない場合はダミーデータで動作確認可能

### 🎯 **4カテゴリ自動分類**
- **仕事**：会議、資料、報告など
- **家のこと**：買い物、掃除、料理など
- **自分のこと**：ジム、勉強、趣味など
- **その他**：上記以外

ChatGPTが自動判定します。

### 📋 **タスク管理機能**
- 各タスクに「今日」「今週」「そのうち」を設定
- フィルター機能（すべて/今日/今週/そのうち）
- チェックボックスで完了→自動削除
- 音声入力のたびに新規リスト作成（追加ではなく上書き）

### 💫 **バブル破裂エフェクト**
タスク完了時に12個のカラフルなバブルが飛び散る美しいアニメーション

### 🎨 **カスタマイズ機能**
- カテゴリ名の編集可能
- 8色から選べるカラー設定
- デフォルトはグレーブルー調

### 💾 **データ永続化**
ローカルストレージで自動保存、ブラウザを閉じても安心

## 🔑 セットアップ

### **1. OpenAI APIキーの取得**
1. [OpenAI Platform](https://platform.openai.com/api-keys) にアクセス
2. アカウント作成（既にある場合はログイン）
3. 「Create new secret key」でAPIキーを生成
4. APIキーをコピー（`sk-...`で始まる文字列）

### **2. APIキーの設定**
1. アプリを開く
2. 左下の「API設定」ボタンをクリック
3. APIキーを入力して「保存」
4. APIキーはブラウザのローカルストレージに安全に保存されます

### **3. 使い方**
1. 「音声入力を開始」ボタンをクリック
2. やることを自然に話す（例：「明日の会議の資料を準備して、牛乳を買いに行って、ジムに行く予定」）
3. もう一度クリックして停止
4. ChatGPTが自動的にタスクを抽出してリスト化！

## 🔒 セキュリティとプライバシー

- APIキーはブラウザのローカルストレージに保存され、外部には一切送信されません
- 音声データはOpenAI APIにのみ送信され、タスク抽出後は保持されません
- タスクデータは全てローカルに保存され、クラウドには送信されません

## 💡 ChatGPT APIについて

- 使用モデル：`gpt-4o-mini`（高速・低コスト）
- 1回の音声入力あたり約0.1円程度
- 月に数百回使っても数十円程度の費用
